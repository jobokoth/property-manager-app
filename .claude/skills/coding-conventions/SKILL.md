# Coding Conventions
- Always update and use the Latest version of Laravel https://github.com/laravel/laravel/blob/12.x/CHANGELOG.md
- Always use MySQL not SQL Lite
- Always use Latest version of Filament for Admin https://github.com/filamentphp/filament/tree/5.x
- For Roles & Permissions use Spatie https://github.com/spatie/laravel-permission
- Always include pagination for list endpoints
- Version APIs in the URL path (/v1/, /v2/)
